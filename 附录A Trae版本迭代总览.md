自发布以来，Trae 经历了多次重要的版本迭代，每次迭代都带来了显著的功能增强和体验优化。以下是 Trae 的主要版本更新记录：

| 版本号 | 发布时间 | 核心更新 | 影响范围 |
| --- | --- | --- | --- |
| v0.1.0 | 2024年初 | 首次发布，基础代码补全和对话功能 | 个人开发者 |
| v0.2.0 | 2024年Q1 | 引入 MTC 模式，支持多轮对话和文档生成 | 内容创作者 |
| v0.3.0 | 2024年Q2 | 引入 Code 模式，支持项目级代码操作 | 全栈开发者 |
| v0.5.0 | 2024年Q2 | 引入 SOLO Builder，支持自然语言生成项目 | 初创团队 |
| v0.7.0 | 2024年Q3 | 引入 SOLO Coder，支持上下文感知的功能开发 | 专业开发者 |
| v1.0.0 | 2024年Q3 | 正式版发布，全面优化稳定性和用户体验 | 所有用户 |
| v1.2.0 | 2024年Q4 | 团队协作功能，知识库共享，代码审查 | 企业团队 |
| v1.5.0 | 2025年Q1 | 多模态支持（设计稿转代码），插件系统预览 | 设计师+开发者 |
| v1.8.0 | 2025年Q2 | AI 原生开发范式，意图驱动开发 | 前沿开发者 |
| v2.0.0 | 2025年Q3 | 全面升级，企业级功能，安全合规 | 企业用户 |
| v2.5.0 | 2025年Q4 | 垂直领域深化，行业解决方案 | 行业用户 |

从版本迭代可以看出，Trae 的发展路径清晰而坚定：从基础的代码补全到高级的自主开发，从个人工具到团队平台，从通用能力到垂直深化。每一次版本更新都在扩展 Trae 的能力边界，使其能够服务更广泛的用户群体和更丰富的应用场景。
