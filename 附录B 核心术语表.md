以下是本书中涉及的核心术语及其详细解释，帮助读者快速理解关键概念。

| 术语 | 英文 | 解释 |
| --- | --- | --- |
| AI编程 | AI Programming | 利用人工智能技术辅助或自动化软件开发的实践，包括代码生成、补全、审查、调试等 |
| 大语言模型 | LLM (Large Language Model) | 经过大规模文本数据训练的深度学习模型，具备自然语言理解和生成能力 |
| 提示词工程 | Prompt Engineering | 设计和优化输入给 AI 模型的提示词，以获得高质量输出的技术和方法 |
| 代码补全 | Code Completion | AI 根据上下文自动补全正在编写的代码片段，是 AI 编程最基础的功能 |
| MTC模式 | MTC Mode | Trae 的对话模式，支持多轮对话、文档生成和知识问答，适合需求分析和方案设计 |
| Code模式 | Code Mode | Trae 的代码模式，支持项目级代码操作，包括文件读写、代码修改和终端命令执行 |
| SOLO Builder | SOLO Builder | Trae 的项目生成功能，根据自然语言描述自动生成完整的项目代码 |
| SOLO Coder | SOLO Coder | Trae 的智能编码功能，在已有项目基础上进行上下文感知的功能开发 |
| 流式输出 | Streaming Output | AI 生成内容时逐步返回结果，而不是等待全部生成完成后一次性返回 |
| Token | Token | 大语言模型处理文本的基本单位，中文通常一个字对应1-2个Token |
| 上下文窗口 | Context Window | AI 模型在一次对话中能够处理的最大文本量，限制了输入和输出的总长度 |
| RAG | RAG (Retrieval-Augmented Generation) | 检索增强生成技术，通过检索外部知识来增强 AI 的生成能力 |
| Agent | Agent | AI 代理，能够自主感知环境、做出决策并执行动作的智能体 |
| 温度参数 | Temperature | 控制 AI 输出随机性的参数，值越高输出越有创造性，值越低输出越确定 |
| 微调 | Fine-tuning | 在预训练模型的基础上，使用特定领域的数据进行进一步训练，提升模型在特定任务上的表现 |
